# coleforcongress.com

Campaign website

## Developing locally

1. Clone down the repo
2. `script/bootstrap` to install dependencies
3. `script/server` to run the site
4. visit [localhost:4000](http://localhost:4000) in your browser

## Testing

`script/cibuild`
